About Game

WordCraft is a multiplayer game. Invite your friends and play! Players must enter as many number of words they can from the given random letters within 20 seconds.

Game Rules

  • You can reuse the given letters
  • Words must be a valid one
  • Do not repeat words
  • Game doesnot accept single letters

Game Steps

  • Go to Want to Play section
  • Give your name and enter GameRoom
  • To start the game type /ready in chat room
  • 5 Seconds count down will be displayed
  • Start entering your words in the input box

Now its time to play!
